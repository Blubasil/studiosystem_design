# Studio System Website

A Next.js website built with React 19, Tailwind CSS, and Radix UI components.

## Run Locally

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).

### Other Commands

```bash
# Build for production
pnpm build

# Start production server
pnpm start

# Run linter
pnpm lint
```

## Deployment

Deployed on [Vercel](https://vercel.com).
